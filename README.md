- user สามารถvoteเห็นด้วยกับความน่าเหลือเชื่อเพื่อดันให้หัวข้อขึ้นไปโชว์ด้านบนได้
- user สามารถsearchหาเรื่องน่าเหลือเชื่อจากtagต่างๆได้
